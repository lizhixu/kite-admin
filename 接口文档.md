# 认证

## POST 登录

POST /auth/login

> Body 请求参数

```json
{
  "username": "admin",
  "password": "123456",
  "captcha": "12gl"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» username|body|string| 是 |none|
|» password|body|string| 是 |none|
|» captcha|body|string| 是 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "accessToken": "access-token:admin:super-admin"
  },
  "originUrl": "/auth/login"
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|
